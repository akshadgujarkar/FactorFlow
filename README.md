# 🏢 SocietyDAO  
### A Decentralized Society Treasury & Governance System

> A futuristic Web3-based housing society platform for transparent fund management, automated maintenance collection, and decentralized governance.

---

## 🌐 Overview

**SocietyDAO** is a decentralized application (dApp) designed to manage residential society finances using blockchain principles.

It transforms traditional housing society management into a **trustless, transparent, and automated system** where residents collectively:

- Manage treasury funds
- Pay monthly maintenance fees
- Create & vote on spending proposals
- Track financial activities in real time

Built with a **DAO-first architecture**, SocietyDAO removes centralized control and ensures full financial transparency.

---

## 🚀 Key Features

### 💰 Transparent Treasury System
- On-chain fund tracking
- Real-time balance updates
- Deposit history with wallet-level traceability

### 🏠 Automated Maintenance Collection
- Monthly maintenance fee system per flat
- Smart contract-based payment tracking
- Dynamic late fee calculation (increasing daily after due date)
- Payment status: Paid / Pending / Overdue

### ⏳ Penalty System (Smart Contract Driven)
- Base maintenance fee (fixed monthly)
- Daily increment after deadline
- Fully automated enforcement (no admin intervention)

### 🗳 DAO Governance
- Create proposals for society expenses
- Vote using wallet-based voting power
- YES / NO voting system
- Quorum-based decision making
- Execution tracking

### 📊 Analytics Dashboard
- Voting analytics (Yes vs No distribution)
- Maintenance collection rate
- Participation metrics
- Quorum progress tracking

### 📜 Activity Transparency
- Full transaction history
- Proposal lifecycle tracking
- Maintenance payment logs
- Wallet-level traceability
